# accessibility-learning

アクセシビリティ対応UI サンプル作成用のリポジトリです。

### 開発

ローカルで確認する際は、VSCodeのLiver Serverを起動します。

| Command          | Description                          |
| :--------------- | :----------------------------------- |
| npm start        | 開発を開始するためのコマンドです。   |
| npm run build    | 本番環境用のファイルをビルドします。 |
| npm run prettify | コードをフォーマットします。         |
